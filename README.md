📦 Inventaris App

Aplikasi Inventaris Barang Sekolah buatan **Frido Avan Almuzaqi**.  
Dibuat pakai **Next.js + TailwindCSS + ShadCN UI** buat ngatur peminjaman barang dengan stok otomatis berkurang.
______________________________________

🚀 Cara Jalankan

```bash
npm install
npm run dev
Buka di browser 👉 http://localhost:3000

🧩 Fitur Utama
- Input nama, barang, jumlah, dan tanggal
- Stok barang otomatis berkurang setelah dipinjam
- Riwayat peminjaman tersimpan di localStorage
- Edit & hapus data peminjaman
- UI elegan pakai Tailwind + ShadCN Card

🧠 Teknologi
- Next.js 14
- Tailwind CSS
- ShadCN UI
- TypeScript
- LocalStorage
